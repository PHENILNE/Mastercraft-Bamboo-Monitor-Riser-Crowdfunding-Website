Mastercraft Bamboo Monitor Riser – Crowdfunding Website

A crowdfunding landing page for the Mastercraft Bamboo Monitor Riser,  
a beautifully handcrafted monitor stand designed to reduce neck and eye strain.  
This project focuses on clean UI design and interactive frontend functionality.

Features :
- Product showcase and description
- Pledge and reward sections
- Interactive buttons and states
- Responsive layout for mobile and desktop
- Clean and modern user interface

Technologies Used :
- HTML- page structure  
- CSS– styling and responsive layout
